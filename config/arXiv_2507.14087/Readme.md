# Global Bayesian Analysis of J/ψ Photoproduction on Proton and Lead Targets

## eprint: [link](https://arxiv.org/abs/2507.14087)

This folder contains parameter files to compute the vortex ring observable
as proposed in the paper, arXiv:2507.14087.
